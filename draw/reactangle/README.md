# Draw Rectangle (K230)

## 📋 Описание

Примеры использования функции `draw_rectangle()` для рисования прямоугольников на плате Yahboom K230.
Основано на официальной PDF документации.

### Что такое draw_rectangle?

`draw_rectangle()` — это функция для рисования прямоугольников на изображении. 
Она позволяет создавать как контуры, так и залитые цветом прямоугольники.

## 🧠 API

### Синтаксис

```python
img.draw_rectangle(x, y, w, h, color=(R,G,B), thickness=1, fill=False)
```

### Параметры

| Параметр | Описание | Пример |
|----------|----------|--------|
| `x` | Координата X левого верхнего угла (в пикселях) | `100` |
| `y` | Координата Y левого верхнего угла (в пикселях) | `100` |
| `w` | Ширина прямоугольника (в пикселях) | `200` |
| `h` | Высота прямоугольника (в пикселях) | `150` |
| `color` | Цвет в формате RGB (красный, зелёный, синий) | `(255,0,0)` - красный |
| `thickness` | Толщина линии контура (в пикселях) | `2` |
| `fill` | Заполнить прямоугольник цветом (`True` или `False`) | `True` |

### Примеры цветов

- `(255, 0, 0)` — Красный
- `(0, 255, 0)` — Зелёный
- `(0, 0, 255)` — Синий
- `(255, 255, 0)` — Жёлтый
- `(0, 191, 255)` — Голубой
- `(135, 206, 235)` — Светло-голубой
- `(255, 255, 255)` — Белый
- `(0, 0, 0)` — Чёрный

## 🎥 Pipeline (Цепочка обработки)

### Для примера с камерой:

```
Камера → Снимок (snapshot) → Рисование прямоугольников → Дисплей
```

1. **Sensor** — камера захватывает изображение
2. **snapshot()** — получение кадра из камеры
3. **draw_rectangle()** — рисование прямоугольников на кадре
4. **Display.show_image()** — вывод изображения на экран

## 👨‍💻 Для программиста

### Советы по использованию:

1. **Координаты**: Система координат начинается с левого верхнего угла (0, 0)
2. **Комбинирование**: Можно комбинировать `draw_rectangle()` с другими функциями:
   - `draw_circle()` — рисование кругов
   - `draw_line()` — рисование линий
   - `draw_string()` — добавление текста

3. **Производительность**: При работе с камерой старайтесь минимизировать количество операций рисования в цикле

### Пример комбинации:

```python
# Нарисовать прямоугольник
img.draw_rectangle(50, 50, 100, 100, color=(255,0,0), thickness=2)

# Добавить круг внутри
img.draw_circle(100, 100, 30, color=(0,255,0), fill=True)

# Добавить текст
img.draw_string(50, 200, "Hello!", color=(255,255,255))
```

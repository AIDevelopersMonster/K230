# Draw Arrow (K230)

## 📋 Описание

Примеры использования функции `draw_arrow()` для рисования стрелок на плате Yahboom K230.  
Эти скрипты демонстрируют, как рисовать стрелки на статическом изображении и поверх видеопотока с камеры.

Основано на официальной PDF-документации K230.

## 🔧 Требования

- Плата: **Yahboom K230**
- Дисплей: **ST7701** (640x480)
- Камера (для примера с камерой)

## 🧠 API

### Рисование стрелки

```python
img.draw_arrow(x0, y0, x1, y1, color=(R, G, B), thickness=1)
```

**Параметры:**
| Параметр | Описание |
|----------|----------|
| `x0`, `y0` | Координаты начала стрелки (верхний левый угол = 0,0) |
| `x1`, `y1` | Координаты конца стрелки (куда указывает стрелка) |
| `color` | Цвет стрелки в формате RGB, например `(255, 0, 0)` — красный |
| `thickness` | Толщина линии стрелки в пикселях (по умолчанию 1) |

### Рисование текста

```python
img.draw_string(x, y, "Текст", color=(R, G, B))
```

## 📁 Примеры

### 01_draw_arrow_demo.py — Статические стрелки

**Описание:**  
Скрипт создаёт изображение с белым фоном и рисует несколько стрелок разных цветов и толщины.

**Что делает:**
- Создаёт белый фон
- Рисует главную горизонтальную стрелку
- Добавляет дополнительные стрелки выше и ниже
- Рисует диагональные стрелки
- Отображает результат на дисплее

**Как запустить:**
```bash
python 01_draw_arrow_demo.py
```

### 02_draw_arrow_camera.py — Стрелки поверх камеры

**Описание:**  
Скрипт захватывает изображение с камеры и рисует поверх него цветные стрелки и текст.

**Что делает:**
- Инициализирует камеру
- Захватывает кадр
- Рисует красную и зелёную стрелки
- Добавляет надпись "Draw Arrow"
- Отображает результат на дисплее в реальном времени

**Как запустить:**
```bash
python 02_draw_arrow_camera.py
```

## 🎥 Pipeline (поток данных)

### Для примера с камерой:
```
Камера → Snapshot (захват кадра) → draw_arrow (рисование) → Display (экран)
```

### Для статического примера:
```
Создание изображения → Рисование фона → draw_arrow → Display (экран)
```

## 👨‍💻 Для разработчиков

**Полезные советы:**
- Координаты начинаются с верхнего левого угла экрана (0, 0)
- X растёт вправо, Y растёт вниз
- Цвета задаются в формате RGB (0-255 для каждого канала)
- Можно комбинировать `draw_arrow()` с другими функциями рисования

**Идеи для проектов:**
- Визуализация направления движения робота
- Указатели для системы навигации
- Стрелки для отображения углов и направлений
- UI элементы для сенсорного интерфейса

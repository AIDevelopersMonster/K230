# Draw Text (K230)

## 📋 Описание

В этой папке находятся примеры отрисовки текста на плате Yahboom K230.
Скрипты демонстрируют использование методов `draw_string()` и `draw_string_advanced()`
для вывода текста на экран или поверх изображения с камеры.

Все комментарии в коде написаны на русском языке, чтобы новичкам было проще
понять, как работает каждый элемент скрипта.

## 🚀 Быстрый старт

1. Подключите плату K230 к компьютеру
2. Откройте любой из примеров в IDE
3. Запустите скрипт и наблюдайте результат на экране

## 🧠 API методы

### draw_string()
Базовый метод для отрисовки текста.

```python
img.draw_string(x, y, text, color=(R, G, B), scale=1)
```

**Параметры:**
- `x`, `y` — координаты левого верхнего угла текста
- `text` — строка для отображения
- `color` — цвет текста в формате (Красный, Зелёный, Синий), значения от 0 до 255
- `scale` — масштаб шрифта (1 = базовый размер)

### draw_string_advanced()
Расширенный метод с поддержкой Unicode (русский, китайский и др.).

```python
img.draw_string_advanced(x, y, size, text, color=(R, G, B))
```

**Параметры:**
- `x`, `y` — координаты левого верхнего угла текста
- `size` — размер шрифта в пикселях
- `text` — строка для отображения (поддерживает Unicode)
- `color` — цвет текста в формате (Красный, Зелёный, Синий)

## 📁 Примеры

### 01_draw_text_demo.py
**Что делает:** Создаёт белый фон и отображает текст на английском и китайском языках.

**Особенности:**
- Использует `draw_string_advanced()` для поддержки Unicode
- Статичное изображение (без камеры)
- Демонстрация работы с цветом текста

### 02_draw_text_camera.py
**Что делает:** Захватывает изображение с камеры и добавляет текстовые надписи поверх него.

**Особенности:**
- Использует `draw_string()` для быстрой отрисовки
- Работает в реальном времени с камерой
- Показывает, как накладывать текст на видеопоток

## 🐛 Устранение проблем

| Проблема | Решение |
|----------|---------|
| Текст не отображается | Проверьте инициализацию Display и MediaManager |
| Неправильные цвета | Убедитесь, что формат цвета (R, G, B) правильный |
| Камера не работает | Проверьте подключение и инициализацию сенсора |

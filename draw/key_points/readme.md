# Рисование ключевых точек (K230)

## 📋 Описание

Примеры использования функций `draw_keypoints()` и `find_keypoints()` для детектирования и отображения ключевых точек на плате Yahboom K230.

Основано на официальной PDF-документации K230.

## 🎯 Для новичков

Этот пример покажет вам:
- Что такое ключевые точки (keypoints) и как их использовать
- Как детектировать ключевые точки на изображении с камеры
- Как рисовать найденные ключевые точки на дисплее
- Как определить область интереса (ROI) для поиска точек
- Как комбинировать видеопоток с камеры и графику

## 🧠 API функции

### find_keypoints()

Поиск ключевых точек на изображении:

```python
keypoints = img.find_keypoints(threshold=30, scale_factor=1.2, max_keypoints=30, roi=None)
```

**Параметры:**
- `threshold` — порог чувствительности детектора (чем меньше, тем больше точек найдётся)
- `scale_factor` — масштаб для поиска точек разного размера (обычно 1.1-1.3)
- `max_keypoints` — максимальное количество найденных точек
- `roi` — область интереса (Region of Interest), формат: (x, y, ширина, высота)

**Возвращает:** объект keypoints или None, если точки не найдены

### draw_keypoints()

Рисование найденных ключевых точек:

```python
img.draw_keypoints(keypoints, color=(R,G,B), size=8, thickness=2, fill=True)
```

**Параметры:**
- `keypoints` — объект с найденными ключевыми точками (результат find_keypoints)
- `color` — цвет точек в формате (Красный, Зелёный, Синий), значения от 0 до 255
- `size` — размер каждой ключевой точки в пикселях
- `thickness` — толщина линий обводки
- `fill` — если `True`, точки будут закрашены сплошным цветом

### draw_cross()

Рисование крестика в указанных координатах:

```python
img.draw_cross(x, y, color=(R,G,B), size=10)
```

**Параметры:**
- `x` — координата центра крестика по оси X (горизонталь)
- `y` — координата центра крестика по оси Y (вертикаль)
- `color` — цвет крестика в формате (Красный, Зелёный, Синий)
- `size` — размер крестика в пикселях

**Примеры цветов:**
- `(255, 0, 0)` — красный
- `(0, 255, 0)` — зелёный
- `(0, 0, 255)` — синий
- `(255, 255, 0)` — жёлтый
- `(255, 255, 255)` — белый
- `(0, 0, 0)` — чёрный

## 📁 Примеры скриптов

### 01_draw_keypoints_demo.py — Демонстрация ключевых точек

**Что делает:**
Создаёт статичное изображение с белым фоном и рисует на нём ряд красных крестиков, имитируя ключевые точки для детектирования объектов.

**Чему научитесь:**
- Создавать изображение с заданным разрешением и форматом цвета
- Рисовать простые фигуры (прямоугольники, крестики)
- Использовать цикл для рисования множества одинаковых элементов
- Инициализировать дисплей и отображать на нём изображение

**Запуск:**
```bash
python 01_draw_keypoints_demo.py
```

### 02_draw_keypoints_camera.py — Ключевые точки с камеры

**Что делает:**
Захватывает видеопоток с камеры в реальном времени и:
- Отображает цветное изображение с камеры
- Ищет ключевые точки в указанной области (ROI) на чёрно-белом изображении
- Рисует найденные ключевые точки красным цветом поверх видеопотока
- Выделяет область поиска светло-голубой рамкой

**Чему научитесь:**
- Инициализировать камеру на K230 с двумя потоками (цветной и ч/б)
- Получать кадры с камеры через `sensor.snapshot()`
- Использовать `find_keypoints()` для детектирования особенностей изображения
- Рисовать графику поверх реального видеопотока
- Работать с областью интереса (ROI)
- Создавать приложения компьютерного зрения в реальном времени

**Запуск:**
```bash
python 02_draw_keypoints_camera.py
```

## 💡 Что такое ключевые точки?

**Ключевые точки (keypoints)** — это характерные особенности изображения, которые можно надёжно обнаружить при различных условиях:
- Углы объектов
- Границы между областями разной яркости
- Текстуры и паттерны
- Особо контрастные участки

**Применение:**
- Распознавание объектов
- Слежение за движением
- Стереозрение (определение глубины)
- Дополненная реальность
- Навигация роботов

## 👤 Для пользователя

1. Подключите плату K230 к компьютеру
2. Загрузите один из скриптов на плату
3. Запустите скрипт через IDE или напрямую
4. Наблюдайте результат на дисплее

**Для скрипта с камерой:**
- Направьте камеру на объект с выраженной текстурой или углами
- Ключевые точки будут автоматически найдены и подсвечены

## 👨‍💻 Для разработчика

**Полезные советы:**
- Координаты (0, 0) находятся в левом верхнем углу экрана
- Для центра экрана используйте (WIDTH//2, HEIGHT//2)
- Область интереса (ROI) помогает ускорить поиск и снизить ложные срабатывания
- Параметр `threshold` влияет на чувствительность: меньшее значение = больше точек
- Комбинируйте `draw_keypoints()` с другими функциями:
  - `draw_rectangle()` — выделение областей
  - `draw_string()` — текстовые подписи
  - `draw_line()` — соединение точек

**Идеи для проектов:**
- Детектор углов объектов
- Система слежения за движущимся объектом
- Распознавание жестов
- Визуальный одометр для робота
- Интерактивная доска с определением касаний

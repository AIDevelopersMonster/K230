# Draw examples for Yahboom K230

## Что находится в этой папке

Папка `draw/` содержит обучающие примеры по графическим функциям K230 Vision Module. Эти функции позволяют рисовать поверх изображения простые элементы интерфейса: линии, круги, прямоугольники, стрелки, прицелы, текст и ключевые точки.

Главная идея всех примеров одинаковая:

```text
image → draw_* → display
```

Для camera-примеров pipeline такой:

```text
camera → snapshot → draw_* → Display.show_image()
```

То есть сначала мы получаем изображение, затем рисуем на нём графику, а потом выводим результат на экран или в IDE.

## Основные функции рисования

### Line

```python
img.draw_line(x0, y0, x1, y1, color=(255, 0, 0), thickness=2)
```

Рисует линию от начальной точки `(x0, y0)` до конечной точки `(x1, y1)`.

Используется для:
- направляющих;
- сеток;
- рамок;
- простых UI-элементов.

---

### Circle

```python
img.draw_circle(x, y, radius, color=(0, 255, 0), thickness=2, fill=False)
```

Рисует круг с центром `(x, y)` и радиусом `radius`.

Используется для:
- маркеров;
- зон интереса;
- декоративных элементов;
- прицелов.

---

### Rectangle

```python
img.draw_rectangle(x, y, w, h, color=(0, 191, 255), thickness=2, fill=False)
```

Рисует прямоугольник. Координаты `(x, y)` — верхний левый угол, `w` и `h` — ширина и высота.

Используется для:
- bounding box;
- выделения объектов;
- интерфейсных блоков;
- ROI-областей.

---

### Ellipse

```python
img.draw_ellipse(cx, cy, rx, ry, rot, color=(255, 0, 0), thickness=2)
```

Рисует эллипс с центром `(cx, cy)`, радиусами `rx`, `ry` и поворотом `rot`.

Используется для:
- областей внимания;
- визуальных эффектов;
- округлых UI-элементов.

---

### Arrow

```python
img.draw_arrow(x0, y0, x1, y1, color=(255, 255, 0), thickness=2)
```

Рисует стрелку от точки `(x0, y0)` к точке `(x1, y1)`.

Используется для:
- направления движения;
- навигации;
- визуализации трекинга;
- подсказок пользователю.

---

### Crosshair

```python
img.draw_cross(x, y, color=(255, 0, 0), size=20, thickness=2)
```

Рисует крест / прицел в точке `(x, y)`.

Используется для:
- прицелов;
- маркеров точек;
- выделения центра;
- Computer Vision overlay.

---

### Text

```python
img.draw_string(x, y, "Text", color=(255, 255, 0), scale=2)
img.draw_string_advanced(x, y, 30, "Hello", color=(0, 191, 255))
```

Рисует текст поверх изображения. `draw_string_advanced()` рекомендуется для более гибкого вывода текста и поддержки разных языков.

Используется для:
- подписей;
- FPS;
- статуса модели;
- названий объектов;
- интерфейса пользователя.

---

### Key Points

```python
keypoints = img_g.find_keypoints(threshold=30, scale_factor=1.2, max_keypoints=30, roi=roi)
img.draw_keypoints(keypoints, color=(255, 0, 0), size=8, thickness=2, fill=True)
```

Ключевые точки — это важные места изображения: углы, пересечения, контрастные участки. Они нужны для трекинга, сопоставления изображений и Computer Vision.

Важно: `find_keypoints()` работает с grayscale-изображением.

## Для программиста

Все примеры показывают один общий принцип:

```python
img = sensor.snapshot(...)
img.draw_*(...)
Display.show_image(img)
```

Вы можете комбинировать функции:

```python
img.draw_rectangle(100, 100, 200, 120, color=(255, 0, 0), thickness=2)
img.draw_string(100, 80, "Object", color=(255, 255, 0), scale=2)
img.draw_cross(200, 160, color=(0, 255, 0), size=20)
```

Так создаются HUD-интерфейсы, разметка объектов и визуализация результатов AI-моделей.

## Чтение и запись файлов

### Сохранение изображения

```python
img.save("/data/result.jpg")
```

Так можно сохранить кадр с уже нарисованными линиями, текстом, рамками или keypoints.

### Загрузка изображения

```python
img = image.Image("/data/result.jpg")
Display.show_image(img)
```

Так можно открыть ранее сохранённое изображение и вывести его на экран.

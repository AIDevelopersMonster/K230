# Draw Line (K230)

## 📋 Что это

Примеры использования функции `draw_line()` для рисования линий на изображении и поверх изображения с камеры.

Основано на документации:
fileciteturn0file0

## 🧠 API

```python
img.draw_line(x0, y0, x1, y1, color=(R,G,B), thickness=1)
```

| Параметр | Описание |
|----------|---------|
| x0, y0 | Начальная точка |
| x1, y1 | Конечная точка |
| color | Цвет линии (RGB) |
| thickness | Толщина линии |

## 📁 Примеры

### 01_draw_line_logo.py
- Рисует слово "Yahboom" линиями
- Без камеры
- Подходит для изучения API

### 02_draw_line_camera.py
- Рисует линии поверх камеры
- Показывает реальное применение

## 🎥 Как работает с камерой

Пайплайн:

```
camera → snapshot → draw_line → display
```

Это подтверждается примерами камеры в репозитории, где:
- `sensor.snapshot()` получает кадр
- `Display.show_image()` выводит его fileciteturn1file0

## 💾 Чтение и запись файлов

### Сохранение изображения

```python
img.save("/data/snapshot/test.jpg")
```

Пример из репозитория:
- сохранение каждого 100 кадра fileciteturn4file0

### Чтение файла

```python
img = image.Image("/data/snapshot/test.jpg")
Display.show_image(img)
```

## 🚀 Быстрый старт

1. Подключите камеру

## 💡 Для программистов

- Все рисование происходит над объектом `image`
- Можно комбинировать с `draw_string`, `draw_rectangle`
- Работает в реальном времени

## 👤 Для пользователей

- Просто запустите пример

# Draw Line (K230)

## 📋 Что это

Примеры использования функции `draw_line()` для рисования линий на изображении и поверх изображения с камеры.

**Автор:** AIDevelopersMonster  
**Плата:** Yahboom K230  
**GitHub:** https://github.com/AIDevelopersMonster/K230

## 🧠 API

```python
img.draw_line(x0, y0, x1, y1, color=(R,G,B), thickness=1)
```

| Параметр | Описание |
|----------|---------|
| x0, y0 | Начальная точка линии (координаты X и Y) |
| x1, y1 | Конечная точка линии (координаты X и Y) |
| color | Цвет линии в формате RGB (красный, зелёный, синий) |
| thickness | Толщина линии в пикселях (по умолчанию 1) |

## 🎥 Как работает с камерой

Пайплайн обработки изображения:

```
Камера → snapshot (захват кадра) → draw_line (рисование) → display (вывод на экран)
```

Пошагово:
1. `sensor.reset()` - сброс настроек камеры
2. `sensor.set_framesize()` - установка разрешения кадра
3. `sensor.set_pixformat()` - установка формата цвета
4. `sensor.snapshot()` - получение текущего кадра
5. `img.draw_line()` - рисование линий на кадре
6. `Display.show_image()` - вывод результата на дисплей

## 💡 Для программистов

- Все операции рисования выполняются над объектом `image.Image`
- Можно комбинировать с другими функциями: `draw_string()`, `draw_rectangle()`, `draw_circle()`
- Рисование работает в реальном времени без задержек
- Координатная система: (0,0) - левый верхний угол экрана

## 👤 Для пользователей

- Экспериментируйте с координатами для создания своих узоров

## ⚙️ Настройка параметров

### Изменение цвета линии
```python
# Красный цвет
color = (255, 0, 0)
# Зелёный цвет
color = (0, 255, 0)
# Синий цвет
color = (0, 0, 255)
# Белый цвет
color = (255, 255, 255)
```

### Изменение толщины линии
```python
# Тонкая линия
thickness = 1
# Средняя линия
thickness = 3
# Толстая линия
thickness = 5
```

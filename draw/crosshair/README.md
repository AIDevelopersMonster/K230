# Draw Crosshair (K230)

## 📋 Описание

Примеры использования функции `draw_cross()` для рисования прицелов (крестиков) на плате Yahboom K230.

Этот проект демонстрирует два подхода:
1. **Статичное изображение** - рисование прицела на белом фоне
2. **Камера в реальном времени** - наложение прицела на видеопоток с камеры

Основано на официальной PDF документации K230.

## 🔧 Требования

- Плата Yahboom K230
- Дисплей ST7701 (для примеров с выводом на экран)
- Камера (для примера 02)

## 🧠 API Функции

### draw_cross() - Рисование креста/прицела

```python
img.draw_cross(x, y, color=(R, G, B), size=5, thickness=1)
```

**Параметры:**
- `x` - координата X центра креста (горизонтальная позиция)
- `y` - координата Y центра креста (вертикальная позиция)
- `color` - цвет креста в формате (Red, Green, Blue), значения от 0 до 255
- `size` - размер креста в пикселях (длина каждой из 4 линий от центра)
- `thickness` - толщина линий креста в пикселях

**Примеры цветов:**
- `(255, 0, 0)` - красный
- `(0, 255, 0)` - зелёный
- `(0, 0, 255)` - синий
- `(255, 255, 0)` - жёлтый
- `(0, 191, 255)` - голубой

## 📁 Примеры

### 01_draw_crosshair_demo.py - Статичный прицел

**Что делает:**
- Создаёт белый фон
- Рисует большой центральный крест голубого цвета
- Добавляет 8 маленьких крестиков по внутреннему кругу
- Добавляет 12 крестиков по внешнему кругу
- Отображает результат на дисплее

**Особенности:**
- Использует тригонометрию для расчёта позиций по кругу
- Демонстрирует разные размеры и цвета крестов

**Запуск:**
```bash
python 01_draw_crosshair_demo.py
```

### 02_draw_crosshair_camera.py - Прицел поверх камеры

**Что делает:**
- Захватывает изображение с камеры в реальном времени
- Рисует красный прицел в центре экрана
- Добавляет две зелёные точки прицеливания
- Отображает текст "Draw Crosshair"
- Показывает обработанное изображение на дисплее

**Особенности:**
- Работает в реальном времени (60 FPS)
- Подходит для систем прицеливания
- Обрабатывает нажатие кнопки выхода

**Запуск:**
```bash
python 02_draw_crosshair_camera.py
```

## 🎥 Pipeline обработки

### Для примера с камерой:
```
Камера → Захват кадра (snapshot) → Рисование крестов (draw_cross) → Дисплей
```

### Для статичного примера:
```
Создание изображения → Рисование фона → Рисование крестов → Дисплей
```

## 👤 Для начинающих

1. **Изучите координаты**: Экран 640x480, центр находится в точке (320, 240)
2. **Экспериментируйте с цветами**: Меняйте значения RGB для получения разных цветов
3. **Регулируйте размер**: Параметр `size` контролирует размер креста
4. **Попробуйте толщины**: Параметр `thickness` меняет толщину линий

## 👨‍💻 Для разработчиков

- Используйте целочисленное деление `//` для координат центра
- Комбинируйте несколько `draw_cross()` для создания сложных прицелов
- Не забывайте освобождать ресурсы в блоке `finally`
- Проверяйте точку выхода через `os.exitpoint()` в циклах

## 🛠️ Модули

- `media.sensor` - работа с камерой
- `media.display` - вывод на дисплей
- `media.media` - медиа-менеджер
- `image` - обработка изображений

# Draw Circle (K230)

## 📋 Описание

Примеры использования функции `draw_circle()` для рисования кругов на плате Yahboom K230.

Основано на официальной PDF-документации K230.

## 🎯 Для новичков

Этот пример покажет вам:
- Как рисовать круги разных размеров и цветов
- Как использовать координаты X, Y и радиус
- Как комбинировать несколько кругов для создания сложных фигур
- Как рисовать поверх видеопотока с камеры

## 🧠 API функции draw_circle()

```python
img.draw_circle(x, y, radius, color=(R,G,B), thickness=1, fill=False)
```

**Параметры:**
- `x` — координата центра круга по оси X (горизонталь)
- `y` — координата центра круга по оси Y (вертикаль)
- `radius` — радиус круга в пикселях
- `color` — цвет круга в формате (Красный, Зелёный, Синий), значения от 0 до 255
- `thickness` — толщина линии обводки (по умолчанию 1 пиксель)
- `fill` — если `True`, круг будет закрашен сплошным цветом

**Примеры цветов:**
- `(255, 0, 0)` — красный
- `(0, 255, 0)` — зелёный
- `(0, 0, 255)` — синий
- `(255, 255, 0)` — жёлтый
- `(255, 255, 255)` — белый
- `(0, 0, 0)` — чёрный

## 📁 Примеры скриптов

### 01_draw_circle_demo.py — Механическое колесо

**Что делает:**
Рисует статичное изображение "механического колеса" с использованием нескольких кругов:
- Два внешних кольца разного размера
- Центральный узел (ступица)
- 8 декоративных элементов по кругу (как болты на автомобильном колесе)

**Чему научитесь:**
- Рисовать круги в центре экрана (координаты 320, 240 для разрешения 640x480)
- Использовать параметр `fill=True` для закрашивания круга
- Использовать математику (тригонометрию) для расчёта позиций элементов по кругу
- Комбинировать круги разного размера для создания сложных фигур

**Запуск:**
```bash
python 01_draw_circle_demo.py
```

### 02_draw_circle_camera.py — Круги поверх камеры

**Что делает:**
Захватывает видеопоток с камеры и рисует поверх него:
- Красный круг радиусом 100 пикселей
- Зелёный круг радиусом 50 пикселей
- Текстовую подпись "Draw Circle + Camera"

**Чему научитесь:**
- Инициализировать камеру на K230
- Получать кадры с камеры через `sensor.snapshot()`
- Рисовать графику поверх реального видеопотока
- Создавать интерактивные приложения с дополненной реальностью

**Запуск:**
```bash
python 02_draw_circle_camera.py
```

## 👨‍💻 Для разработчика

**Полезные советы:**
- Координаты (0, 0) находятся в левом верхнем углу экрана
- Для центра экрана используйте (WIDTH//2, HEIGHT//2)
- Комбинируйте `draw_circle()` с другими функциями рисования:
  - `draw_line()` — линии
  - `draw_rectangle()` — прямоугольники
  - `draw_string()` — текст
- Экспериментируйте с прозрачностью и толщиной линий

**Идеи для проектов:**
- Прицел для камеры
- Индикаторы на приборной панели
- Простые игры (мишени, кнопки)
- Визуализация данных (круговые диаграммы)

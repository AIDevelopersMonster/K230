# Draw Ellipse (K230)

## 📋 Описание

Примеры использования функции `draw_ellipse()` для рисования эллипсов на плате Yahboom K230.

Основано на официальной PDF документации K230.

## 🎯 Что вы узнаете

- Как создавать изображение и рисовать на нём эллипсы
- Как захватывать изображение с камеры и добавлять графические элементы
- Как работать с дисплеем и медиа-системой K230

## 🧠 API функции draw_ellipse()

```python
img.draw_ellipse(cx, cy, rx, ry, rot, color=(R,G,B), thickness=1)
```

**Параметры:**
- `cx`, `cy` - координаты центра эллипса (в пикселях)
- `rx` - радиус эллипса по оси X (горизонтальный размер)
- `ry` - радиус эллипса по оси Y (вертикальный размер)
- `rot` - угол поворота эллипса в градусах (0-360)
- `color` - цвет в формате RGB кортеж (R, G, B), где каждый компонент от 0 до 255
- `thickness` - толщина линии обводки в пикселях (по умолчанию 1)

## 📁 Примеры

### 01_draw_ellipse_demo.py - Случайные эллипсы

**Что делает:**
- Создаёт белый фон на экране
- Рисует 10 случайных эллипсов с разными параметрами:
  - Случайная позиция центра
  - Случайные размеры (радиусы)
  - Случайный угол поворота
  - Случайный яркий цвет

**Запуск:**
```bash
python 01_draw_ellipse_demo.py
```

### 02_draw_ellipse_camera.py - Эллипсы поверх изображения с камеры

**Что делает:**
- Захватывает изображение с камеры в реальном времени
- Рисует два концентрических эллипса в центре экрана:
  - Большой красный эллипс (120x60 пикселей)
  - Маленький зелёный эллипс (60x30 пикселей)
- Добавляет надпись "Draw Ellipse" жёлтым цветом

**Запуск:**
```bash
python 02_draw_ellipse_camera.py
```

## 🎥 Pipeline обработки изображений

```
Камера → Snapshot (захват кадра) → draw_ellipse (рисование) → Display (вывод на экран)
```

## 👨‍💻 Для разработчиков

**Полезные советы:**
- Используйте `(WIDTH//2, HEIGHT//2)` для получения центра экрана
- Радиусы `rx` и `ry` определяют размер эллипса по горизонтали и вертикали
- Если `rx == ry`, получится круг
- Угол поворота `rot` измеряется в градусах против часовой стрелки
- Цвета задаются в формате RGB: `(255,0,0)` - красный, `(0,255,0)` - зелёный, `(0,0,255)` - синий

**Форматы цветов:**
- `(255, 0, 0)` - Красный
- `(0, 255, 0)` - Зелёный
- `(0, 0, 255)` - Синий
- `(255, 255, 0)` - Жёлтый
- `(255, 255, 255)` - Белый
- `(0, 0, 0)` - Чёрный

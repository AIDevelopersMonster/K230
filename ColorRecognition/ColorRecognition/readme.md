# Color Recognition (K230)

## 📌 Описание

Проект демонстрирует распознавание цветов с помощью платы **Yahboom K230** и фреймворка **CanMV**.

Модуль показывает как распознавать цвета с помощью метода:

```python
img.find_blobs()
```

Используется **LAB цветовое пространство**, которое лучше подходит для выделения цветов при различном освещении.

---

## 🧠 Как это работает

1. **Камера получает изображение** — сенсор захватывает кадр с разрешением 640x480.
2. **Применяется LAB threshold** — задаются пороги для выделения нужного цвета.
3. **Находятся цветовые области** — метод `find_blobs()` ищет пиксели в заданном диапазоне.
4. **Рисуется прямоугольник и центр** — найденные объекты выделяются на экране.

### Что такое LAB?

- **L** — светимость (от 0 до 100)
- **A** — от зелёного до красного
- **B** — от синего до жёлтого

## ⚙️ Настройка цвета

Для точной настройки порогов цвета используйте встроенный инструмент CanMV:

**Tools → Machine Vision → Threshold Editor**

1. Подключите K230 к компьютеру.
2. Откройте Threshold Editor.
3. Наведите камеру на объект нужного цвета.
4. Подберите значения L, A, B.
5. Скопируйте полученные значения и добавьте в список `THRESHOLDS` в коде.

Пример порога:
```python
THRESHOLDS = [
    (0, 66, 7, 127, 3, 127),          # красный
    (42, 100, -128, -17, 6, 66),      # зелёный
    (43, 99, -43, -4, -56, -7),       # синий
]
```

## 🚀 Запуск скрипта

1. Откройте CanMV IDE.
2. Подключите плату K230 по USB.
3. Откройте файл `.py` в редакторе.
4. Нажмите **Run** (зелёная кнопка).
5. Наведите камеру на объект нужного цвета.

## 🔧 Полезные советы

- **Улучшите освещение** — при плохом свете цвета определяются хуже.
- **Уберите блики** — отражения могут исказить цвет.
- **Настройте пороги** — каждый объект может требовать индивидуальной настройки.
- **Используйте_merge=True** — для объединения близких областей одного цвета.

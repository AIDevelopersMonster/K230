# Linear Regression (K230)

## 📌 Описание

Модуль показывает как использовать функцию:

```python
img.get_regression()
```

для поиска линии.

📖 Основано на примере из PDF (line patrol) — камера смотрит на черную линию на белом фоне. fileciteturn25file0

---

## 🧪 Примеры

| Файл | Описание |
|------|----------|
| 01 | базовый поиск линии |
| 02 | с бинаризацией |
| 03 | расчёт ошибки (для робота) |

---

## 🧠 Как работает

1. камера получает изображение
2. выделяются пиксели (threshold)
3. строится линия методом least squares
4. получаем направление движения

---

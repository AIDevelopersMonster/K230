# Color Recognition (K230) | Распознавание цветов

## 📌 Описание / Description

Модуль показывает, как распознавать цвета с помощью камеры K230, используя LAB цветовое пространство.

Этот модуль содержит два примера:
- **01_color_recognition_camera.py** - Базовое распознавание одного цвета с отображением рамки и центра объекта
- **02_multi_color_tracking.py** - Одновременное распознавание нескольких цветов

---

## 🔧 Как это работает / How it works

1. **Получение изображения** - Камера делает снимок в разрешении 640x480 пикселей
2. **LAB фильтрация** - Применяются пороги LAB для выделения нужного цвета
3. **Поиск объектов** - Функция `img.find_blobs()` находит области заданного цвета
4. **Отрисовка** - Вокруг найденных объектов рисуется прямоугольник и крестик в центре
5. **Отображение** - Результат показывается на дисплее с указанием FPS

### Что такое LAB цветовое пространство?

LAB состоит из трёх компонентов:
- **L (Lightness)** - светлота от 0 (чёрный) до 100 (белый)
- **A** - зелёно-красная компонента от -128 (зелёный) до 127 (красный)
- **B** - сине-жёлтая компонента от -128 (синий) до 127 (жёлтый)

## ⚙️ Настройка цвета / Color Setup

### Использование Threshold Editor

1. Откройте CanMV IDE
2. Перейдите в **Tools → Machine Vision → Threshold Editor**
3. Наведите камеру на объект нужного цвета
4. Подберите значения L, A, B так, чтобы объект выделялся
5. Скопируйте полученные значения
6. Добавьте их в список `THRESHOLDS` в коде

### Структура порога цвета

```python
(L Min, L Max, A Min, A Max, B Min, B Max)
```

Пример для красного цвета:
```python
(0, 66, 7, 127, 3, 127)
```

## 🚀 Быстрый старт / Quick Start

### Для примера 01 (один цвет):

1. Откройте файл `01_color_recognition_camera.py` в CanMV IDE
2. Выберите нужный цвет, изменив переменную `COLOR_INDEX`:
   - `0` - красный
   - `1` - зелёный
   - `2` - синий
   - `3` - голубой/YAHBOOM
3. Подключите плату K230 к компьютеру через USB
4. Запустите скрипт кнопкой **Run**

### Для примера 02 (несколько цветов):

1. Откройте файл `02_multi_color_tracking.py` в CanMV IDE
2. При необходимости настройте пороги цветов в списке `THRESHOLDS`
3. Подключите плату K230 к компьютеру через USB
4. Запустите скрипт кнопкой **Run**

## 💡 Советы для новичков / Tips for Beginners

### Если цвет определяется плохо:

1. **Улучшите освещение** - убедитесь, что объект хорошо освещён
2. **Уберите блики** - избегайте отражений света от объекта
3. **Настройте пороги** - используйте Threshold Editor для точной настройки
4. **Измените thresholds** - подберите значения под ваше освещение

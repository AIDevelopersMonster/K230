# Обнаружение кругов (Circle Detection) для Yahboom K230

## 📋 Описание

В этой папке находятся скрипты для обнаружения кругов с помощью камеры на плате **Yahboom K230** (CanMV). Скрипты захватывают изображение с камеры, находят круги и отображают результат на LCD-экране и в терминале.

## 📁 Файлы

| Файл | Описание |
|------|----------|
| `01_circle_detection_lcd_light_dark.py` | Обнаружение **чёрных кругов на белом фоне**. Выводит количество кругов, их радиус и координаты. |
| `02_circle_detection_dark_mode.py` | Обнаружение **белых кругов на чёрном фоне**. Использует бинаризацию для улучшения контраста. |
| `img/` | Папка с изображениями (черный и белый круги для тестирования). |

## 🚀 Как использовать

### Требования
- Плата **Yahboom K230**
- Подключенная камера (ID=2)
- LCD-дисплей ST7701 (640x480)
- CanMV IDE или прямое подключение к плате

### Запуск скрипта

1. Подключите плату K230 к компьютеру через USB.
2. Откройте CanMV IDE.
3. Скопируйте один из скриптов (`01_...py` или `02_...py`) в IDE.
4. Нажмите кнопку **Run** (или F5).

### Настройка параметров

В начале каждого скрипта есть раздел **НАСТРОЙКИ ПАРАМЕТРОВ**, где можно изменить:

```python
PICTURE_WIDTH = 400      # Ширина изображения с камеры
PICTURE_HEIGHT = 240     # Высота изображения с камеры
CIRCLE_THRESHOLD = 3500  # Порог обнаружения кругов
```

**Совет:** 
- Уменьшите `CIRCLE_THRESHOLD`, если круги не обнаруживаются.
- Увеличьте `CIRCLE_THRESHOLD`, если много ложных срабатываний.

## 🔍 Как это работает

### Основной алгоритм:

1. **Инициализация камеры** — настройка разрешения и формата пикселей.
2. **Инициализация дисплея** — подготовка LCD-экрана для вывода.
3. **Захват кадра** — получение изображения с камеры.
4. **Поиск кругов** — функция `find_circles()` анализирует изображение.
5. **Отрисовка результатов** — найденные круги обводятся, ставится крест в центре.
6. **Вывод информации** — количество кругов и FPS показываются на экране и в терминале.

### Для тёмного фона (`02_...py`):
Дополнительно применяется **бинаризация** изображения:
```python
img.binary([(200, 255)])  # Оставляем только яркие пиксели
```
Это помогает лучше выделить белые круги на чёрном фоне.

## 📊 Выводимая информация

На экране вы увидите:
- **Контур круга** — голубой (для светлого фона) или белый (для тёмного).
- **Крест в центре** — красный маркер центра круга.
- **Подпись** — номер круга и его радиус (например, "C1 R25").
- **Статистика** — количество кругов и FPS в левом верхнем углу.

В терминале выводится:
```
[Circle Detection Start]
count: 2
Circle 1 (x, y, r)
Circle 2 (x, y, r)
FPS: 30.5
```

## ⚠️ Возможные проблемы

| Проблема | Решение |
|----------|---------|
| Круги не обнаруживаются | Уменьшите `CIRCLE_THRESHOLD` до 2000-2500 |
| Много ложных срабатываний | Увеличьте `CIRCLE_THRESHOLD` до 4000-5000 |
| Изображение не центрировано | Проверьте значения `DISPLAY_WIDTH` и `DISPLAY_HEIGHT` |
| Скрипт не запускается | Убедитесь, что камера подключена к порту ID=2 |

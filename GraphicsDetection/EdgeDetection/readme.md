# Edge Detection (K230)

## Описание
Демонстрация обнаружения границ объектов с помощью K230.

Используется функция:
image.find_edges()

Согласно документации:
- Используется алгоритм Canny
- Работает только с grayscale изображением

## Примеры

### 1. Камера (LCD)
01_edge_detection_lcd.py

### 2. Работа с файлами
02_edge_detection_image_io.py

## Чтение файла
img = image.Image("/sd/input.jpg")

## Запись файла
img.save("/sd/output_edges.jpg")

## Настройки
threshold = (50, 80)

- первый параметр — нижний порог
- второй — верхний

## Результат
Белые линии — это границы объектов

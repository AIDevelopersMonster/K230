# Camera Display на K230

## Что это

Camera display — это вывод изображения с камеры на экран.

## Как работает

камера → sensor.snapshot() → Display.show_image()

## Примеры

01 - базовый вывод
02 - наложение текста
03 - сохранение кадров

## Важно

- формат RGB565
- разрешение 640x480
- sensor.run() обязателен

## Применение

- AI
- видеонаблюдение
- интерфейсы

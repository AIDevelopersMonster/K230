# Touch Display на K230

## Что это такое

Touch Display — это экран, который реагирует на касания пользователя.

Это позволяет:
- создавать интерфейсы
- управлять устройством пальцем
- делать интерактивные приложения

## Основная идея

```text
палец → координаты → обработка → действие
```

## Основные шаги

1. Инициализация touch
2. Чтение координат
3. Обработка
4. Отображение результата

## Основные функции

- Touch.init()
- Touch.read()

## Примеры

| Файл | Описание |
|------|--------|
| 01 | базовое чтение касания |
| 02 | рисование пальцем |
| 03 | кнопка на экране |

## Что важно

- Touch.read() возвращает координаты (x, y)
- нужно использовать цикл
- важно обновлять экран

## Вывод

Touch позволяет сделать интерфейс как в смартфоне

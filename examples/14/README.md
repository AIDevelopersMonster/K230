# Многопоточность на K230

## Что такое многопоточность

Многопоточность — это выполнение нескольких задач "одновременно".

На K230 используется модуль `_thread` для запуска нескольких потоков.

## Основные идеи

- Каждый поток выполняет свою функцию
- Потоки работают параллельно (логически)
- Вывод в терминале перемешивается

## Важно

- Нужно использовать `sleep()` или `sleep_ms()`
- Без этого поток может занять CPU полностью

## Основные функции

- `_thread.start_new_thread(func, args)` — запуск потока
- `_thread.get_ident()` — ID потока
- `_thread.allocate_lock()` — создание lock

## Итог

Многопоточность = несколько задач одновременно

Но требует контроля (sleep, lock)

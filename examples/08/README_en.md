# PWM Examples for K230

PWM (Pulse Width Modulation) allows control of power using duty cycle.

## Basics

- control LED brightness
- generate signals
- audio generation

## Pins

- IO42 → PWM0
- IO43 → PWM1

## Examples

| File | Description |
|------|-------------|
| 01_pwm_basic.py | Basic PWM |
| 02_pwm_led_fade.py | LED fade |
| 03_pwm_button_control.py | Button control |
| 04_pwm_uart_control.py | UART control |
| 05_pwm_music.py | PWM sound |

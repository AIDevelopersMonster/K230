# PWM примеры для K230

## 📚 Что такое PWM?

**PWM (Pulse Width Modulation)** или **ШИМ (Широтно-Импульсная Модуляция)** — это метод управления мощностью сигнала путём изменения длительности импульсов при постоянной частоте.

### Простыми словами:

Представьте, что вы быстро включаете и выключаете свет. Если вы включаете свет на 50% времени и выключаете на 50% времени, глаз воспринимает это как половинную яркость. PWM работает точно так же!

### Ключевые понятия:

1. **Частота (freq)** — сколько раз в секунду сигнал включается и выключается. Измеряется в Герцах (Гц).
   - Пример: 1000 Гц = 1000 циклов в секунду

2. **Скважность (duty cycle)** — процент времени, когда сигнал находится во включённом состоянии.
   - 0% — сигнал всегда выключен
   - 50% — сигнал включён половину времени
   - 100% — сигнал всегда включен

### Визуализация PWM:

```
100% duty: ████████████████████ (всегда включено)
50% duty:  ████████░░░░░░░░░░░░ (половина времени)
25% duty:  ████░░░░░░░░░░░░░░░░ (четверть времени)
0% duty:   ░░░░░░░░░░░░░░░░░░░░ (всегда выключено)
```

## 🔧 Для чего используется PWM?

1. **Управление яркостью светодиодов** — изменение скважности меняет видимую яркость
2. **Управление скоростью моторов** — больше мощность = быстрее вращение
3. **Генерация звуковых сигналов** — изменение частоты создаёт разные ноты
4. **Сервоприводы** — точное управление положением

## 📌 Пины на плате Yahboom K230

| Пин | Функция |
|-----|---------|
| IO42 | PWM0 |
| IO43 | PWM1 |

## 🚀 Как запускать скрипты?

### Способ 1: Через Thonny IDE (рекомендуется для новичков)

1. Установите [Thonny IDE](https://thonny.org/)
2. Подключите плату K230 к компьютеру через USB
3. В Thonny выберите: `Tools → Options → Interpreter → MicroPython (K230)`
4. Откройте файл `.py` и нажмите кнопку "Run" (▶️)

### Способ 2: Через командную строку (REPL)

1. Подключитесь к плате через последовательный порт (например, PuTTY или screen)
2. Скорость подключения: 115200 бод
3. Скопируйте код скрипта в терминал или загрузите файл на плату
4. Запустите командой: `import имя_файла` (без .py)

### Способ 3: Загрузка файла на плату

1. Сохраните скрипт на плату в основную директорию
2. Переименуйте в `main.py` для автоматического запуска при включении
3. Или запускайте вручную через `import`

## 💡 Советы для новичков

1. **Начните с первого примера** — он самый простой и показывает основы
2. **Экспериментируйте с параметрами** — меняйте `freq` и `duty`, смотрите результат
3. **Подключите светодиод** — к пину IO42 через резистор (220 Ом) для визуального эффекта
4. **Читайте комментарии в коде** — они подробно объясняют каждую строку
5. **Не бойтесь ошибаться** — если что-то не работает, проверьте подключение и параметры

# PWM примеры для K230

PWM — это широтно-импульсная модуляция.

## Основы

- управление яркостью
- управление скоростью
- генерация сигналов

## Пины

- IO42 → PWM0
- IO43 → PWM1

| 03_pwm_button_control.py | Кнопка → PWM |

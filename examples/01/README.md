# MicroPython в REPL: примеры с пояснениями

Этот файл содержит простые примеры для **REPL-консоли** на K230 / CanMV. Их можно вводить по одной строке или небольшими блоками, чтобы быстро проверить, как работает MicroPython.

> Если REPL не отвечает сразу, попробуйте сначала нажать **Ctrl + C**, а затем **Enter**.

---

## Зачем использовать REPL

REPL удобен для:

- проверки отдельных команд
- знакомства с синтаксисом MicroPython
- отладки простых идей
- проверки модулей и объектов
- тестов перед переносом кода в `.py` файл

---

## 1. Проверка, что REPL работает

Введите:

```python
print("Hello, K230!")
```

Ожидаемый результат:

```python
Hello, K230!
```

Это самый простой способ проверить, что консоль работает и код выполняется.

---

## 2. Простая арифметика

```python
print(1 + 2)
print(10 - 3)
print(4 * 5)
print(8 / 2)
```

Это полезно, чтобы убедиться, что интерпретатор выполняет выражения правильно.

### Целочисленное деление

```python
print(7 // 2)
```

`//` возвращает целое число. Это полезно, например, для координат пикселей.

### Остаток от деления

```python
print(7 % 3)
```

Оператор `%` возвращает остаток. Он полезен, например, для проверки чётности.

---

## 3. Переменные

```python
x = 10
y = 20
print(x + y)
```

Здесь:

- `x` и `y` — переменные
- `print(x + y)` выводит сумму

REPL хорошо подходит для мгновенной проверки значений.

---

## 4. Строки

```python
name = "K230"
print("Плата:", name)
print(name.lower())
print(len(name))
```

Что здесь происходит:

- хранится текст в переменной
- `.lower()` переводит строку в нижний регистр
- `len(...)` показывает длину строки

---

## 5. Логические проверки

```python
print(5 > 3)
print(5 == 3)
print(5 != 3)
```

REPL удобен для проверки условий и логики перед использованием их в программе.

---

## 6. Условие `if`

```python
temperature = 65

if temperature > 60:
    print("Температура повышенная")
else:
    print("Температура в норме")
```

Важно помнить:

- в REPL блоки вводятся с отступами
- после блока обычно нужно нажать Enter ещё раз, чтобы завершить ввод

---

## 7. Цикл `for`

```python
for i in range(3):
    print(i)
```

Ожидаемый вывод:

```python
0
1
2
```

Это простой способ проверить циклы.

---

## 8. Цикл `while`

```python
count = 0
while count < 3:
    print(count)
    count += 1
```

Такой пример полезен, чтобы понять повторение действий.

> Если случайно запустили бесконечный цикл, нажмите **Ctrl + C**.

---

## 9. Списки

```python
items = [10, 20, 30]
print(items)
print(items[0])
items.append(40)
print(items)
```

Здесь можно проверить:

- создание списка
- доступ по индексу
- добавление элементов

---

## 10. Словари

```python
camera = {
    "width": 640,
    "height": 480,
    "fps": 30
}

print(camera["width"])
print(camera.get("fps"))
```

Словари удобны для хранения настроек и параметров.

---

## 11. Функции

```python
def add(a, b):
    return a + b

print(add(2, 3))
```

REPL отлично подходит, чтобы быстро проверить, как работает функция.

---

## 12. Импорт модулей

```python
import os
print(os.listdir())
```

Этот пример полезен, чтобы проверить:

- работает ли импорт
- доступна ли файловая система
- какие файлы есть на устройстве

---

## 13. Просмотр информации о системе

```python
import sys
print(sys.version)
```

Так можно быстро посмотреть версию MicroPython.

---

## 14. Проверка свободной памяти

```python
import gc
gc.collect()
print(gc.mem_free())
```

Это полезно на embedded-устройствах, потому что память ограничена.

---

## 15. Работа с файлами

```python
with open("test.txt", "w") as f:
    f.write("hello")

with open("test.txt", "r") as f:
    print(f.read())
```

Такой тест помогает понять, доступна ли запись и чтение файлов.

---

## 16. Простая проверка исключений

```python
try:
    print(10 / 0)
except Exception as e:
    print("Ошибка:", e)
```

REPL полезен для проверки обработки ошибок без запуска большого файла.

---

## 17. Полезные команды REPL

### Остановить текущий код

```text
Ctrl + C
```

### Мягкий перезапуск интерпретатора

```text
Ctrl + D
```

### Вызвать справку

```python
help()
```

## 18. Мини-практика для K230

Пример с вычислением центра изображения:

```python
width = 640
height = 480
center_x = width // 2
center_y = height // 2
print(center_x, center_y)
```

Это полезно для задач камеры и компьютерного зрения.

---

## 19. Как работать в REPL удобнее

Полезный порядок:

1. Подключиться к плате
2. Нажать **Ctrl + C**, если консоль молчит
3. Проверить `print("hello")`
4. Пробовать короткие команды
5. Когда код заработал — переносить его в `.py` файл

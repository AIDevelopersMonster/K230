# Camera K230

## Что это

Камера позволяет получать изображение и сохранять его.

## Как работает

camera → snapshot → display → save

## Основные функции

- sensor.snapshot()
- Display.show_image()
- img.save()

## Примеры

- preview
- snapshot
- overlay

## Важно

- RGB565
- CAM_CHN_ID_1
- sensor.run()

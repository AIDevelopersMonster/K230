# Camera K230 - Примеры работы с камерой

## 📋 Что это такое?

Эта папка содержит примеры работы с камерой на плате Yahboom K230. Здесь вы найдёте три базовых скрипта, которые демонстрируют основные возможности камеры: вывод изображения на дисплей, сохранение снимков и наложение текста на изображение.

## 🎯 Описание примеров

### 01_camera_preview.py - Базовый предпросмотр
**Что делает:** Захватывает изображение с камеры и отображает его на дисплее в реальном времени.

**Основные функции:**
- `sensor.reset()` - сброс настроек камеры
- `sensor.set_framesize()` - установка разрешения (640x480)
- `sensor.set_pixformat()` - установка формата цвета (RGB565)
- `sensor.snapshot()` - захват одного кадра
- `Display.show_image()` - отображение на дисплее

### 02_camera_snapshot_save.py - Сохранение снимков
**Что делает:** Отображает изображение с камеры и автоматически сохраняет каждый 100-й кадр в файл JPG.

**Особенности:**
- Снимки сохраняются в директорию `/data/snapshot/`
- Имена файлов: 100.jpg, 200.jpg, 300.jpg и т.д.
- Используется задержка 10 мс между кадрами для стабильности

**Основные функции:**
- `img.save(path)` - сохранение изображения в файл
- `time.sleep_ms(10)` - задержка в миллисекундах
- `print()` - вывод информации в консоль

### 03_camera_overlay_info.py - Наложение текста
**Что делает:** Отображает изображение с камеры с наложенным текстом (название и счётчик кадров).

**Особенности:**
- Текст рисуется прямо на изображении до отображения
- Жёлтый текст "Camera" в верхней части
- Зелёный текст со счётчиком кадров

**Основные функции:**
- `img.draw_string(x, y, text, color)` - рисование текста
- Координаты указываются в пикселях от левого верхнего угла
- Цвет задаётся в формате RGB кортежем (R, G, B)

## 📝 Основные компоненты

### Sensor (Камера)
- **Инициализация:** `sensor = Sensor()`
- **Настройка:** `sensor.reset()`, `sensor.set_framesize()`, `sensor.set_pixformat()`
- **Запуск:** `sensor.run()`
- **Захват кадра:** `sensor.snapshot(chn=CAM_CHN_ID_1)`

### Display (Дисплей)
- **Инициализация:** `Display.init(Display.ST7701, width=640, height=480, to_ide=True)`
- **Отображение:** `Display.show_image(img)`

### MediaManager
- **Инициализация:** `MediaManager.init()` - необходим для работы с медиа-ресурсами

## ⚠️ Важные моменты

1. **Формат изображения:** RGB565 (16-битный цвет)
2. **Канал камеры:** Используйте `CAM_CHN_ID_1` для основного канала
3. **Разрешение:** 640x480 пикселей оптимально для дисплея
4. **Запуск камеры:** Обязательно вызывайте `sensor.run()` перед циклом
5. **Точка выхода:** `os.exitpoint()` нужен для корректного завершения через IDE

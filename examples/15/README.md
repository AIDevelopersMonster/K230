# Чтение и запись файлов на K230

## Что это такое

Чтение и запись файлов — это базовая операция, позволяющая сохранять данные на устройстве и затем их использовать.

Например:
- сохранение данных датчика
- запись логов
- хранение конфигурации

Как указано в документации, все операции (загрузка модулей, сохранение фото) используют файловую систему fileciteturn33file0

## Где хранятся файлы

На K230 используется путь:

```python
/sdcard/
```

Он соответствует папке sdcard в системе fileciteturn33file0

## Основные операции

### Запись
```python
with open('/sdcard/file.txt', 'w') as f:
    f.write('hello')
```

### Чтение
```python
with open('/sdcard/file.txt', 'r') as f:
    data = f.read()
```

## Режимы

- r — чтение
- w — запись (перезапись)
- a — добавление
- rb — бинарный режим

## Почему with лучше

- файл закрывается автоматически
- безопасно при ошибках fileciteturn33file0

## Ограничения

- не читать большие файлы целиком
- ограничение памяти fileciteturn33file0

# Чтение и запись файлов на K230

## 📖 Что это такое

Чтение и запись файлов — это базовая операция, позволяющая сохранять данные на устройстве и затем их использовать.

**Примеры использования:**
- 📊 Сохранение данных с датчиков
- 📝 Запись логов программы
- ⚙️ Хранение конфигурации устройства
- 💾 Сохранение результатов вычислений

Все операции на K230 (загрузка модулей, сохранение фотографий, логирование) используют файловую систему.

## 📁 Где хранятся файлы

На плате Yahboom K230 файлы хранятся по пути:

```python
/sdcard/
```

Этот путь соответствует SD-карте, установленной в плату.

## 🔧 Основные операции

### Запись файла
```python
# Открываем файл в режиме записи ('w' — write)
# Если файл существует — он будет перезаписан
# Если файла нет — он будет создан
with open('/sdcard/file.txt', 'w') as f:
    f.write('Hello K230!')
# Файл автоматически закроется после выхода из блока with
```

### Чтение файла
```python
# Открываем файл в режиме чтения ('r' — read)
with open('/sdcard/file.txt', 'r') as f:
    data = f.read()  # Читаем всё содержимое файла
print(data)
```

### Добавление в файл
```python
# Открываем файл в режиме добавления ('a' — append)
# Данные дописываются в конец файла
with open('/sdcard/file.txt', 'a') as f:
    f.write('Новая строка\n')
```

## 📋 Режимы работы с файлами

| Режим | Описание |
|-------|----------|
| `r` | Чтение (файл должен существовать) |
| `w` | Запись (файл создаётся или перезаписывается) |
| `a` | Добавление (данные пишутся в конец файла) |
| `rb` | Чтение в бинарном режиме (для изображений, аудио) |
| `wb` | Запись в бинарном режиме |

## ❓ Почему конструкция `with` лучше?

```python
# ✅ Правильно (с with):
with open('/sdcard/file.txt', 'r') as f:
    data = f.read()
# Файл закроется автоматически, даже если произойдёт ошибка

# ❌ Неправильно (без with):
f = open('/sdcard/file.txt', 'r')
data = f.read()
f.close()  # Нужно не забыть закрыть вручную!
```

**Преимущества `with`:**
- ✅ Файл закрывается автоматически
- ✅ Безопасно при возникновении ошибок
- ✅ Код чище и понятнее

## ⚠️ Ограничения и рекомендации

1. **Не читайте большие файлы целиком** — память K230 ограничена
   ```python
   # Лучше читать построчно:
   with open('/sdcard/large.txt', 'r') as f:
       for line in f:
           print(line)
   ```

2. **Всегда проверяйте наличие SD-карты** перед записью

3. **Используйте try-except** для обработки ошибок:
   ```python
   try:
       with open('/sdcard/file.txt', 'r') as f:
           data = f.read()
   except FileNotFoundError:
       print('Файл не найден!')
   ```

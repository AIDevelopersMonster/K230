# AES примеры для K230

## Что такое AES

AES — симметричный блочный шифр, который шифрует данные блоками по 16 байт.
Используется один и тот же ключ для шифрования и расшифровки. fileciteturn30file0

## Основные свойства

- Блочный шифр (16 байт)
- Ключ: 16 / 24 / 32 байта
- Обратимая функция (в отличие от SHA)

## Режимы

- ECB — простой, но небезопасный
- CBC — цепочка блоков
- CTR — потоковый режим
- GCM — современный режим с аутентификацией fileciteturn30file0

## Важно

ECB режим небезопасен — одинаковые блоки дают одинаковый результат.

# Buzzer для K230

В этой папке приведены примеры работы с буззером и RGB-светодиодом на плате YAHBOOM K230.

## Описание

Буззер — это компонент, который издаёт звук. На K230 используется **пассивный буззер**, который управляется через PWM (широтно-импульсную модуляцию).

Это означает, что:
- Можно задавать частоту (тон звука)
- Можно управлять длительностью и громкостью
- Можно воспроизводить мелодии и сложные звуковые эффекты

В последних примерах показана синхронизация буззера со встроенным RGB-светодиодом для создания светозвуковых эффектов.

## Быстрый старт

### Минимальный пример

```python
from ybUtils.YbBuzzer import YbBuzzer

buzzer = YbBuzzer()
buzzer.beep()  # Короткий стандартный сигнал
```

### Пользовательский сигнал

```python
from ybUtils.YbBuzzer import YbBuzzer

buzzer = YbBuzzer()

# on(частота_Гц, громкость_%, длительность_сек)
buzzer.on(2000, 50, 0.5)  # Частота 2000 Гц, 50% громкости, 0.5 секунды
```

### Завершение работы

```python
buzzer.off()  # Всегда выключайте буззер после использования
```

## API буззера

### Основные методы

| Метод | Описание | Параметры |
|-------|----------|-----------|
| `beep()` | Короткий стандартный сигнал | - |
| `on(freq, volume, duration)` | Включить звук | `freq` - частота в Гц, `volume` - громкость 0-100%, `duration` - длительность в секундах |
| `off()` | Выключить буззер | - |

### Примеры использования

```python
# Стандартный сигнал
buzzer.beep()

# Звук определённой частоты
buzzer.on(1000, 50, 0.2)  # 1 кГц, средняя громкость, 200 мс

# Тревожный сигнал (серия коротких звуков)
for _ in range(3):
    buzzer.on(1200, 60, 0.1)
    time.sleep(0.1)

# Выключение
buzzer.off()
```

## Синхронизация с RGB

В примерах 5 и 6 показано, как синхронизировать буззер с RGB-светодиодом:

```python
from ybUtils.YbBuzzer import YbBuzzer
from ybUtils.YbRGB import YbRGB

buzzer = YbBuzzer()
rgb = YbRGB()

# Включаем цвет и звук одновременно
rgb.show_rgb((255, 0, 0))  # Красный
buzzer.on(523, 50, 0.3)    # Нота C5

# Выключаем
rgb.show_rgb((0, 0, 0))
buzzer.off()
```

## Важно

1. **Пассивный буззер** требует генерации PWM-сигнала. Звук зависит от частоты:
   - Чем выше частота — тем выше тон
   - Диапазон слышимых частот: примерно 200 Гц – 5000 Гц

2. **Всегда выключайте буззер** после завершения работы:
   ```python
   buzzer.off()
   ```

3. **Используйте паузы** между звуками для лучшей читаемости мелодий:
   ```python
   buzzer.on(523, 50, 0.3)
   time.sleep(0.05)  # Короткая пауза между нотами
   ```

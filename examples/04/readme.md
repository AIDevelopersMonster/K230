# Buzzer для K230

В этой папке приведены примеры работы с буззером на плате YAHBOOM K230.

## Описание

Буззер — это компонент, который издаёт звук.
На K230 используется пассивный буззер, который управляется через PWM.

Это означает, что:
- можно задавать частоту (тон)
- можно управлять длительностью звука

## Примеры

- `04_1_basic_buzzer.py` — базовые сигналы (beep, сигнал тревоги)
- `04_2_melody_buzzer.py` — воспроизведение мелодии

## Быстрый старт

```python
from ybUtils.YbBuzzer import YbBuzzer
buzzer = YbBuzzer()
buzzer.beep()
```

## Управление

Основной метод:

```python
buzzer.on(частота, громкость, длительность)
```

Пример:

```python
buzzer.on(2000, 50, 0.5)
```

## Важно

Пассивный буззер требует генерации сигнала (PWM),
поэтому звук зависит от частоты.

Чем выше частота — тем выше тон.

## Совет

После завершения работы всегда выключайте буззер:

```python
buzzer.off()
```

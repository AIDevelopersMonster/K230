# UART примеры для K230

В этой папке показана работа с UART на K230.

## Основы

UART — это последовательная связь между устройствами.

Подключение:
- TX → RX
- RX → TX
- GND → GND

Скорость: 115200

## Примеры

| Файл | Описание |
|------|----------|
| 01_uart_basic.py | Базовая отправка и приём |
| 02_uart_echo.py | Эхо (повтор сообщений) |
| 03_uart_button.py | Отправка состояния кнопки |
| 04_uart_rgb_control.py | Управление RGB по UART |
| 05_uart_buzzer_music.py | Управление звуком |

## Команды

RGB:
- red
- green
- blue
- off

Buzzer:
- beep
- alarm

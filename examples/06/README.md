# UART примеры для K230

В этой папке собраны примеры работы с **UART** (последовательный порт) на плате K230.  
Эти примеры помогут вам понять, как передавать данные между устройствами.

---

## 📚 Что такое UART?

**UART** (Universal Asynchronous Receiver-Transmitter) — это простой способ обмена данными между двумя устройствами по двум проводам.

### Как это работает:
- Одно устройство отправляет данные через **TX** (передающий пин)
- Другое устройство получает через **RX** (приёмный пин)
- И наоборот — для ответа

### 🔌 Подключение проводов

Для связи двух устройств соедините их **перекрёстно**:

| Устройство 1 | → | Устройство 2 |
|--------------|---|--------------|
| TX           | → | RX           |
| RX           | → | TX           |
| GND          | → | GND          |

⚠️ **Важно:** Не соединяйте TX с TX или RX с RX!

### ⚙️ Настройки связи

Оба устройства должны использовать одинаковые настройки:
- **Скорость (Baudrate):** 115200 бит/сек
- **Биты данных:** 8
- **Стоповый бит:** 1
- **Контроль чётности:** Нет

## 🛠 Что понадобится

- Плата K230
- USB-UART адаптер (например, CP2102, CH340, FTDI)
- Провода для подключения
- Терминальная программа на компьютере:
  - **Windows:** PuTTY, Tera Term, Arduino IDE (Serial Monitor)
  - **Linux/macOS:** `screen`, `minicom`, `picocom`

---

## 🚀 Как запустить примеры

1. **Подключите оборудование:**
   - Соедините K230 с UART-адаптером по схеме выше
   - Подключите адаптер к компьютеру через USB

2. **Узнайте имя порта:**
   - **Windows:** COM3, COM4 и т.д. (диспетчер устройств)
   - **Linux:** `/dev/ttyUSB0`, `/dev/ttyACM0`
   - **macOS:** `/dev/tty.usbserial-*`

3. **Запустите терминал:**
   ```bash
   # Linux/macOS пример
   screen /dev/ttyUSB0 115200
   
   # Или в PuTTY укажите порт и скорость 115200
   ```

4. **Загрузите скрипт на K230** и запустите его

5. **Отправляйте команды** из терминала и наблюдайте результат

## 💡 Примеры команд

### Для `04_uart_rgb_control.py`:
Отправьте в терминал одну из команд:
```
red      → красный свет
green    → зелёный свет
blue     → синий свет
off      → выключить свет
```

### Для `05_uart_buzzer_music.py`:
```
beep     → короткий сигнал
alarm    → тревожный сигнал (3 гудка)
```

## ❓ Частые проблемы

| Проблема | Решение |
|----------|---------|
| Ничего не приходит | Проверьте подключение TX↔RX, убедитесь что скорость 115200 |
| Кракозябры вместо текста | Неправильная скорость (baudrate) на одном из устройств |
| Порт не найден | Проверьте драйверы UART-адаптера, переподключите кабель |

# RGB Lights для K230

В этой папке собраны простые примеры работы со встроенным RGB-светодиодом на плате YAHBOOM K230.

## Что находится в папке

- `03_1_basic_rgb.py` — базовый пример включения цвета и выключения светодиода.
- `03_2_breathing_rgb.py` — эффект плавного «дыхания» с использованием `math.sin()`.
- `03_3_color_cycle_rgb.py` — последовательный перебор нескольких цветов.

## Что нужно для запуска

- плата YAHBOOM K230;
- CanMV IDE;
- библиотека `ybUtils.YbRGB`.

## Быстрый старт

1. Подключите K230 к компьютеру.
2. Откройте CanMV IDE.
3. Создайте новый файл или откройте один из примеров из этой папки.
4. Загрузите код на плату и запустите его.

## Основы управления RGB

Для управления используется объект `YbRGB`:

```python
from ybUtils.YbRGB import YbRGB
rgb = YbRGB()
rgb.show_rgb((255, 0, 0))
```

Цвет задаётся в формате `(R, G, B)`, где каждое значение находится в диапазоне от `0` до `255`.

Примеры:

- `(255, 0, 0)` — красный
- `(0, 255, 0)` — зелёный
- `(0, 0, 255)` — синий
- `(255, 255, 255)` — белый
- `(0, 0, 0)` — выключено

## Важное замечание

Во внутреннем драйвере WS2812B данные могут передаваться в порядке `GRB`, а не `RGB`, но в пользовательских примерах используется обычный вызов `show_rgb((R, G, B))`.

## О чём стоит помнить

- Слишком быстрая смена цветов может выглядеть как мерцание.
- Для плавных эффектов удобно использовать маленькие задержки `time.sleep()`.
- При остановке программы полезно выключать светодиод командой:

```python
rgb.show_rgb((0, 0, 0))
```

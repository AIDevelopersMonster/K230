# Отображение изображений на K230

## 📋 Что это такое

Отображение изображений — это вывод картинки на экран устройства Yahboom K230.

Это используется для:
- 🖼️ Создания графических интерфейсов (UI)
- 📷 Вывода изображений с камеры
- 📊 Визуализации данных и результатов работы алгоритмов
- 🎮 Разработки игр и интерактивных приложений

Изображения загружаются с SD-карты и выводятся на дисплей через контроллер ST7701.

## 📁 Структура файлов

### Основной путь к ресурсам
```python
/sdcard/resources/
```

### Рекомендуемые форматы файлов
| Формат | Поддержка | Рекомендация |
|--------|-----------|--------------|
| PNG    | ✅ Отлично | Рекомендуется (24/32 бит) |
| JPEG   | ⚠️ Ограничено | Может работать хуже |
| BMP    | ✅ Хорошо | Альтернатива PNG |

## ⚠️ Важные требования

- **Формат**: Рекомендуется использовать PNG 24-bit или 32-bit
- **Разрешение**: Максимум 640×480 пикселей (под размер экрана)
- **Цветовая глубина**: PNG 8-bit не поддерживается, используйте 24/32 bit
- **Расположение**: Файлы должны быть на SD-карте в папке `/sdcard/resources/`

## 🔧 Основные шаги работы

1. **Инициализация дисплея** — настройка контроллера и разрешения
2. **Инициализация MediaManager** — подготовка медиа-подсистемы
3. **Загрузка изображения** — чтение файла с SD-карты
4. **Конвертация в RGB888** — преобразование в формат дисплея
5. **Отображение** — вывод изображения на экран

## 💡 Как запустить пример

1. Скопируйте файлы изображений в `/sdcard/resources/`
2. Подключите дисплей к плате K230
3. Запустите скрипт через IDE или консоль

Пример минимального кода:
```python
from media.display import *
from media.media import *
import image

# Инициализация
Display.init(Display.ST7701, width=640, height=480, osd_num=1, to_ide=True)
MediaManager.init()

# Загрузка и отображение
img = image.Image('/sdcard/resources/wp.png', copy_to_fb=True)
img = img.to_rgb888()
Display.show_image(img)
```

## ❌ Частые ошибки и решения

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Изображение не отображается | PNG 8-bit | Конвертируйте в 24/32 bit |
| Чёрный экран | Файл не найден | Проверьте путь `/sdcard/resources/` |
| Искажённые цвета | Нет конвертации | Используйте `to_rgb888()` |
| Программа завершается | Нет цикла | Добавьте `while True: time.sleep(1)` |

## 🔗 Полезные методы

- `image.Image(path, copy_to_fb=True)` — загрузка изображения
- `img.to_rgb888()` — конвертация в RGB888
- `Display.show_image(img)` — отображение на экране
- `img.draw_string_advanced(x, y, size, text, color)` — рисование текста

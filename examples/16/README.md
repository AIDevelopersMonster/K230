# Отображение изображений на K230

## Что это такое

Отображение изображений — это вывод картинки на экран устройства.

Это используется для:
- UI интерфейсов
- вывода изображений с камеры
- визуализации данных

Согласно документации, изображения загружаются с SD-карты и выводятся на дисплей fileciteturn34file0

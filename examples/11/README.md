# FFT (БПФ) для K230

## 📚 Что такое ПФ и БПФ

**Преобразование Фурье (ПФ)** — это математический способ перевести сигнал из временной области в частотную. Другими словами, оно показывает, из каких частот состоит ваш сигнал.

**БПФ (FFT - Fast Fourier Transform)** — это быстрый алгоритм вычисления преобразования Фурье, который позволяет выполнять расчёты в реальном времени на микроконтроллерах.

## 💡 Простыми словами

Представьте, что вы слушаете аккорд на гитаре. Ваш слух автоматически различает отдельные ноты в этом аккорде. FFT делает то же самое с любым сигналом:

- **Вход**: сигнал, меняющийся во времени (как звуковая волна)
- **Выход**: список частот с их амплитудами (какие «ноты» присутствуют и насколько они громкие)

Любой сложный сигнал можно разложить на набор простых синусоид. FFT показывает, какие частоты есть в сигнале и какова их интенсивность.

## 🔧 Применение FFT

- **Анализ звука** — распознавание нот, частотный анализ музыки
- **Фильтрация шума** — выделение полезного сигнала из шума
- **Обработка сигналов** — медицинское оборудование, радиосвязь
- **Спектральный анализ** — визуализация частотного спектра
- **Вибрационный анализ** — диагностика механизмов
- **Распознавание речи** — выделение характерных частот

## 🚀 Быстрый старт

### Запуск базового примера

```python
# Подключитесь к плате K230 и выполните:
python 01_fft_basic.py
```

Вы увидите массив амплитуд для каждой частоты в сигнале.

### Как читать результаты

Результат FFT — это массив чисел, где каждый элемент соответствует амплитуде определённой частоты:
- Первый элемент (индекс 0) — постоянная составляющая (DC offset)
- Остальные элементы — амплитуды возрастающих частот

## ⚙️ Параметры FFT

При создании объекта FFT указываются три параметра:

```python
fft = FFT(arr, N, 0x555)
```

- **arr** — массив с данными сигнала (тип uint16)
- **N** — размер выборки (должен быть степенью двойки: 64, 128, 256...)
- **0x555** — параметр окна Хэмминга для улучшения точности анализа

## 📝 Советы для новичков

1. **Размер выборки**: Используйте степени двойки (64, 128, 256) для корректной работы FFT
2. **Частота дискретизации**: Чем выше частота采样, тем более высокие частоты можно анализировать
3. **Пороговые значения**: Подбирайте экспериментально для вашей задачи
4. **Окно Хэмминга**: Параметр 0x555 уменьшает «утечку» частот и улучшает точность

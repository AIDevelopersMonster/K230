# FFT (БПФ) для K230

## Что такое ПФ и БПФ

Преобразование Фурье (ПФ) — это способ перевести сигнал из временной области в частотную.

БПФ (FFT) — это быстрый алгоритм вычисления ПФ.

## Простыми словами

Любой сигнал можно разложить на набор синусоид.
FFT показывает какие частоты есть в сигнале.

## Применение

- анализ звука
- фильтрация шума
- обработка сигналов
- спектральный анализ

# Camera Image Flip (K230)

## 📋 Что это

Примеры работы с ориентацией изображения камеры на плате Yahboom K230.

**Image flip** — это изменение ориентации изображения камеры:
- **Горизонтальное отражение (hmirror)** — зеркальное отражение слева направо
- **Вертикальный переворот (vflip)** — переворот сверху вниз
- **Двойной переворот** — комбинация обоих эффектов (поворот на 180°)

## 🔧 Как это работает

```
камера → sensor → flip/vflip → snapshot → display
```

Обработка происходит на уровне сенсора камеры, что обеспечивает высокую производительность без дополнительной нагрузки на процессор.

## 📦 Основные функции

| Функция | Описание | Параметр |
|---------|----------|----------|
| `sensor.set_hmirror(True)` | Горизонтальное зеркало | `True` — включить, `False` — выключить |
| `sensor.set_vflip(True)` | Вертикальный переворот | `True` — включить, `False` — выключить |

## 📁 Примеры

| Файл | Описание | Когда использовать |

## ⚠️ Важно

- **Используйте канал 1**: `chn=CAM_CHN_ID_1` — основной канал для обработки изображений
- **Формат цвета**: `RGB565` — 16-битный формат (красный 5 бит, зелёный 6 бит, синий 5 бит)
- **Запуск сенсора**: `sensor.run()` обязателен перед захватом кадров
- **Освобождение ресурсов**: Всегда вызывайте `sensor.stop()`, `Display.deinit()` и `MediaManager.deinit()` при завершении

## 🚀 Быстрый старт

1. Подключите камеру к плате K230
2. Запустите любой пример через IDE
3. Наблюдайте изображение на дисплее

```python
# Минимальный пример
from media.sensor import *
from media.display import *
from media.media import *

sensor = Sensor()
sensor.reset()
sensor.set_framesize(width=640, height=480, chn=CAM_CHN_ID_1)
sensor.set_pixformat(Sensor.RGB565, chn=CAM_CHN_ID_1)
sensor.set_hmirror(True)  # Включаем горизонтальное отражение

Display.init(Display.ST7701, width=640, height=480, to_ide=True)
MediaManager.init()
sensor.run()

while True:
    img = sensor.snapshot(chn=CAM_CHN_ID_1)
    Display.show_image(img)
```

## 💡 Советы

- **Настройка ориентации**: Экспериментируйте с `set_hmirror()` и `set_vflip()` для достижения нужной ориентации
- **Производительность**: Обработка на уровне сенсора не влияет на FPS
- **Отладка**: Используйте `to_ide=True` для вывода изображения в IDE

## 📚 Дополнительная информация

- Все примеры используют разрешение 640x480 пикселей
- Дисплей: ST7701 с разрешением 640x480
- Для корректной работы необходимо инициализировать `MediaManager`

# Camera Image Flip (K230)

## Что это

Image flip — это изменение ориентации изображения камеры:
- горизонтальное отражение
- вертикальный переворот

## Как это работает

камера → sensor → flip → snapshot → display

## Основные функции

- sensor.set_hmirror(True) — горизонтальное зеркало
- sensor.set_vflip(True) — вертикальный переворот

## Примеры

| Файл | Описание |
|------|--------|
| 01 | обычное изображение |
| 02 | горизонтальное зеркало |
| 03 | вертикальный flip |
| 04 | полный разворот |

## Важно

- использовать chn=CAM_CHN_ID_1
- формат RGB565
- sensor.run() обязателен

## Вывод

Flip выполняется на уровне сенсора — это быстро и эффективно

# Примеры K230 (Buttons, RGB LED, Speaker)

В этой папке представлены примеры работы с:

- Кнопкой (GPIO61)
- RGB светодиодом
- Динамиком (buzzer)

## Примеры

| Файл | Описание |
|------|--------|
| 01_button_basic.py | Простое чтение кнопки |
| 02_button_led.py | Кнопка управляет LED |
| 03_rgb_led.py | Управление RGB |
| 04_speaker_basic.py | Работа с динамиком |
| 05_button_rgb_speaker.py | Комбинированный пример |

## Особенности

- Кнопка использует подтяжку вверх
- Нажатие = 0
- Используется библиотека ybUtils.YbKey

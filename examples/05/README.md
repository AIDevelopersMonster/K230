# Примеры K230 (Buttons, RGB LED, Speaker)

В этой папке представлены примеры работы с различными компонентами (кнопка, RGB светодиод, динамик), соединенными с K230. Эти примеры ориентированы на быстрый старт и знак��мство с функциональностью GPIO.

## Требования

- Python 3.x
- Библиотека `ybUtils`
- Оборудование: GPIO61-кнопка, RGB светодиод, динамик

### Установка
1. Убедитесь, что Python установлен на вашем устройстве.
2. Установите библиотеку `ybUtils`:
   ```bash
   pip install ybUtils
   ```

3. Подключите компоненты:
   - Кнопка: GPIO61 с подтяжкой вверх.
   - RGB светодиод: Подключить к GPIO пинам (указать конкретные пины).
   - Динамик: Подключить через резистор.

## Особенности

- Кнопка использует подтяжку вверх.
- Не нажата = логическая "1", нажата = логическая "0".
- Для управления кнопкой используется библиотека `ybUtils.YbKey`.

## Использование

Запустите один из примеров:
```
python 01_button_basic.py
```

Ожидаемое пов��дение:
- Пример 01: Логирует состояние кнопки в консоли.
- Пример 02: Нажатие кнопки будет включать/выключать LED.

## Отладка и подсказки

- Убедитесь, что компоненты подключены к правильным GPIO пинам.
- Если отсутствуют звуки, проверьте подключения динамика.
- Для RGB светодиода проверьте корректность подключения общего анода/катода.

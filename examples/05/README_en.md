# K230 Examples (Buttons, RGB LED, Speaker)

This folder contains examples for:

- Button (GPIO61)
- RGB LED
- Speaker (buzzer)

## Examples

| File | Description |
|------|------------|
| 01_button_basic.py | Basic button reading |
| 02_button_led.py | Button controls LED |
| 03_rgb_led.py | RGB LED control |
| 04_speaker_basic.py | Speaker usage |
| 05_button_rgb_speaker.py | Combined demo |

## Notes

- Button uses pull-up
- Press = 0
- Uses ybUtils.YbKey library

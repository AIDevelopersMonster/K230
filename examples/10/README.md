# Timer примеры для K230

## Что такое Timer

**Timer (Таймер)** — это аппаратный или программный механизм, который позволяет выполнять код через определённые интервалы времени. Таймеры полезны для создания периодических задач без блокировки основного потока программы.

### Основные понятия

- **Timer(-1)** — виртуальный (программный) таймер. Используется когда не критична высокая точность timing'а
- **ONE_SHOT** — однократный режим. Таймер срабатывает один раз и останавливается
- **PERIODIC** — периодический режим. Таймер срабатывает снова и снова через равные интервалы
- **callback (функция обратного вызова)** — функция, которая автоматически вызывается при срабатывании таймера

## Как работает таймер

```python
from machine import Timer

# Создаём таймер
timer = Timer(-1)

# Вариант 1: запуск по периоду (в миллисекундах)
timer.init(period=100, mode=Timer.ONE_SHOT, callback=my_function)

# Вариант 2: запуск по частоте (в Герцах)
timer.init(freq=5, mode=Timer.PERIODIC, callback=my_function)

# Остановка таймера
timer.deinit()
```

## Применение таймеров на практике

- **Опрос датчиков** — периодическое считывание показаний
- **Управление устройствами** — мигание светодиодами, управление моторами
- **Обработка коммуникаций** — проверка UART, I2C, SPI на наличие данных
- **Фоновые задачи** — выполнение действий по расписанию без блокировки основного кода
- **Создание анимаций** — переключение состояний через равные интервалы

## Советы для новичков

1. **Частота таймера** — выбирайте разумную частоту. Слишком высокая (например, 100 Гц) может перегрузить процессор
2. **Callback функции** — делайте их максимально короткими и быстрыми
3. **Глобальные переменные** — для сохранения состояния между вызовами callback используйте `global`
4. **Остановка таймера** — всегда вызывайте `deinit()` когда таймер больше не нужен

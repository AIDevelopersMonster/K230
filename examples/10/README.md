# Timer примеры для K230

## Что такое Timer

Timer — это аппаратный/программный таймер, который позволяет выполнять код по времени.

## Основы (из PDF)

- Timer(-1) — виртуальный таймер
- ONE_SHOT — один раз
- PERIODIC — постоянно
- callback — функция

## Примеры

01 — базовый timer
02 — кнопка
03 — UART
04 — RGB
05 — звук

## Применение

- задачи по времени
- циклические процессы
- управление устройствами
